# Setup

Requirements: Python installed, an Anthropic API key

- Step 1: Get an Anthropic API key from console.anthropic.com
- Step 2: Open a terminal in this folder
- Step 3: Run `python -m http.server 8000`
- Step 4: Open http://localhost:8000/jupiter-rag.html in your browser
- Step 5: Enter your Anthropic API key when prompted on first load
- Step 6: Upload a Jupiter fund factsheet PDF from jupiteram.com

> Note: never open the HTML file directly from your file system — it must be served via the local server or the app will not work
